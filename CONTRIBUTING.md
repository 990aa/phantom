# Contributing to Phantom

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, open a new issue with a clear title and description, including steps to reproduce.

### Suggesting Enhancements
* Open an issue describing the enhancement and why it would be useful.

### Pull Requests
* Fork the repo and create your branch from `main`.
* Ensure your code follows the existing style and conventions.
* Add tests for any new features or bug fixes.
* Ensure the full test suite passes by running `python test_all.py`.
* All PRs require review before being merged into `main`.
